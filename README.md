# miniproject_cars

This is a very simple cars website built with Django.
        
![Screenshot (9)](https://user-images.githubusercontent.com/82138244/135664675-60af1a33-bc4e-43df-8fda-1686d93afad2.png)

Requrement install:

												pip install pillow 
												pip install pyscopg2
